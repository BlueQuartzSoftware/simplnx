# ITK::Salt And Pepper Noise Image Filter (KW)  #


## Group (Subgroup) ##

ITKImageProcessing (ITKImageProcessing)

## Description ##

Alter an image with fixed value impulse noise, often called salt and pepper noise.

Pixel alteration occurs at a user defined probability. Salt and pepper pixel are equally distributed.

\author Gaetan Lehmann

This code was contributed in the Insight Journal paper "Noise
Simulation". https://hdl.handle.net/10380/3158
